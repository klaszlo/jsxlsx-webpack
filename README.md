# jsxlsx-webpack
test repository for js-xlsx and webpack

## Mini tutorial

1) Build client side with webpack

    npm i -D # install the "devdependencies"
    npm start

2) cd into `dist/` directory

    cd dist
    npm i # install the server "dependencies"

3) start the server

    npm run server

4) enjoy:)

    Got to http://localhost:8080
